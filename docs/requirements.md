# نیازمندی‌های سیستم مدیریت خوابگاه دانشجویان

## نیازمندی‌های کاربران

* امکان ثبت پرونده جدید برای دانشجویان
* سیستم رزرو آنلاین اتاق و تخت
* مدیریت اطلاعات خدمات خوابگاه (نظافت، تعمیرات، غذا)
* ثبت تاریخچه استفاده از خدمات

## نیازمندی‌های سیستم

* امنیت داده‌های دانشجویان
* رابط کاربری ساده و قابل استفاده برای کارکنان
* پشتیبان‌گیری خودکار از داده‌ها
