
# طراحی سیستم سامانه مدیریت خوابگاه دانشجویان

## معماری سیستم

* **بخش مدیریت پرونده دانشجویان**
  مدیریت اطلاعات شخصی، وضعیت اقامت و سوابق دانشجو.
* **ماژول رزرو اتاق و تخت**
  ثبت و مدیریت رزرو اتاق‌ها و تخت‌ها برای دانشجویان.
* **سیستم مدیریت خدمات خوابگاه**
  مدیریت درخواست‌های خدماتی مثل نظافت، تعمیرات و غذا.
* **بخش مالی و پرداخت‌ها**
  ثبت و مدیریت پرداخت‌ها و صدور صورتحساب.
